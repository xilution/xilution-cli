# reprovision_instance

## Description

Reprovision a Fox instance.

## Synopsis

```
xln-cli api integration fox reprovision_instance [options]
```

## Options

`--organization_id (string)`

An organization id. Required.

`--instance_id (string)`

A Fox instance ID. Required.
