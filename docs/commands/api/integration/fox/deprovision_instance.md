# deprovision_instance

## Description

Deprovision a Fox instance.

## Synopsis

```
xln-cli api integration fox deprovision_instance [options]
```

## Options

`--organization_id (string)`

An organization id. Required.

`--instance_id (string)`

A Fox instance ID. Required.
