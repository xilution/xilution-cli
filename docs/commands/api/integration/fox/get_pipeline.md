# get_pipeline

## Description

Gets a Fox pipeline.

## Synopsis

```
xln-cli api integration fox get_pipeline [options]
```

## Options

`--organization_id (string)`

An organization id. Required.

`--pipeline_id (string)`

A Fox pipeline ID. Required.

`--output_file (string)`

Path to output file. Written in JSON format.
