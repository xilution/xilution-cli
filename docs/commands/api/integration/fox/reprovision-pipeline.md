# reprovision-pipeline

## Description

Reprovision a Fox pipeline.

## Synopsis

```
xln-cli api integration fox reprovision-pipeline [options]
```

## Options

`--organization-id (string)`

An organization id. Required.

`--pipeline-id (string)`

A Fox pipeline ID. Required.
