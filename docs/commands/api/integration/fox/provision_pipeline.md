# provision_pipeline

## Description

Provision a Fox pipeline.

## Synopsis

```
xln-cli api integration fox provision_pipeline [options]
```

## Options

`--organization_id (string)`

An organization id. Required.

`--pipeline_id (string)`

A Fox pipeline ID. Required.
