# get_instance_status

## Description

Gets a Fox instance's status.

## Synopsis

```
xln-cli api integration fox get_instance_status [options]
```

## Options

`--organization_id (string)`

An organization id. Required.

`--instance_id (string)`

A Fox instance ID. Required.
