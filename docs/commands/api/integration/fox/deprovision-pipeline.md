# deprovision-pipeline

## Description

Deprovision a Fox pipeline.

## Synopsis

```
xln-cli api integration fox deprovision-pipeline [options]
```

## Options

`--organization-id (string)`

An organization id. Required.

`--pipeline-id (string)`

A Fox pipeline ID. Required.
