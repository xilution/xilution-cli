# delete_instance

## Description

Deletes a Fox instance.

## Synopsis

```
xln-cli api integration fox delete_instance [options]
```

## Options

`--organization_id (string)`

An organization id. Required.

`--instance_id (string)`

A Fox instance ID. Required.
