# get-pipeline

## Description

Gets a Fox pipeline.

## Synopsis

```
xln-cli api integration fox get-pipeline [options]
```

## Options

`--organization-id (string)`

An organization id. Required.

`--pipeline-id (string)`

A Fox pipeline ID. Required.

`--output-file (string)`

Path to output file. Written in JSON format.
