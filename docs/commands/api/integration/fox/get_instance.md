# get_instance

## Description

Gets a Fox instance.

## Synopsis

```
xln-cli api integration fox get_instance [options]
```

## Options

`--organization_id (string)`

An organization id. Required.

`--instance_id (string)`

A Fox instance ID. Required.

`--output_file (string)`

Path to output file. Written in JSON format.
