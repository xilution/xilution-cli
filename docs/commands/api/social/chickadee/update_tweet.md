# update_tweet

## Description

Updates a Chickadee tweet.

## Synopsis

```
xln-cli api social chickadee update_tweet [options]
```

## Options

`--input_file (string)`

Path to JSON formatted input file. Required.

`--tweet_id (string)`

A Chickadee tweet ID. Required.

`--output_file (string)`

Path to output file. Written in JSON format.
