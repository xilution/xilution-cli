# get_tweet_event

## Description

Gets a Chickadee tweet event.

## Synopsis

```
xln-cli api social chickadee get_tweet_event [options]
```

## Options

`--tweet_event_id (string)`

A Chickadee tweet event ID. Required.

`--output_file (string)`

Path to output file. Written in JSON format.
