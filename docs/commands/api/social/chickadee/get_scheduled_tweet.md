# get_scheduled_tweet

## Description

Gets a Chickadee scheduled tweet.

## Synopsis

```
xln-cli api social chickadee get_scheduled_tweet [options]
```

## Options

`--scheduled_tweet_id (string)`

A Chickadee scheduled tweet ID. Required.

`--output_file (string)`

Path to output file. Written in JSON format.
