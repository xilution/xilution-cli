# get_tweet

## Description

Gets a Chickadee tweet.

## Synopsis

```
xln-cli api social chickadee get_tweet [options]
```

## Options

`--tweet_id (string)`

A Chickadee tweet ID. Required.

`--output_file (string)`

Path to output file. Written in JSON format.
