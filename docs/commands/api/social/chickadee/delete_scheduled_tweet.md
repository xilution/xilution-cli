# delete_scheduled_tweet

## Description

Deletes a Chickadee scheduled tweet.

## Synopsis

```
xln-cli api social chickadee delete_scheduled_tweet [options]
```

## Options

`--scheduled_tweet_id (string)`

A Chickadee scheduled tweet ID. Required.
