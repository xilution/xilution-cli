# create_tweet

## Description

Creates a Chickadee tweet.

## Synopsis

```
xln-cli api social chickadee create_tweet [options]
```

## Options

`--input_file (string)`

Path to JSON formatted input file. Required.

`--output_file (string)`

Path to output file. Written in JSON format.
