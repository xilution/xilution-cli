# list_twitter_accounts

## Description

Lists Chickadee twitter accounts.

## Synopsis

```
xln-cli api social chickadee list_twitter_accounts [options]
```

## Options

`--page_number (number)`

The page number. Required.

`--page_size (number)`

The page size. Required.
