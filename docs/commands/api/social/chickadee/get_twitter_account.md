# get_twitter_account

## Description

Gets a Chickadee twitter account.

## Synopsis

```
xln-cli api social chickadee get_twitter_account [options]
```

## Options

`--twitter_account_id (string)`

A Chickadee twitter account ID. Required.

`--output_file (string)`

Path to output file. Written in JSON format.
