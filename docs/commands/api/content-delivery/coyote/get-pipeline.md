# get-pipeline

## Description

Gets a Coyote pipeline.

## Synopsis

```
xln-cli api content-delivery coyote get-pipeline [options]
```

## Options

`--organization-id (string)`

An organization id. Required.

`--pipeline-id (string)`

A Coyote pipeline ID. Required.

`--output-file (string)`

Path to output file. Written in JSON format.
