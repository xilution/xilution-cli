# provision-pipeline

## Description

Provision a Coyote pipeline.

## Synopsis

```
xln-cli api content-delivery coyote provision-pipeline [options]
```

## Options

`--organization-id (string)`

An organization id. Required.

`--pipeline-id (string)`

A Coyote pipeline ID. Required.
