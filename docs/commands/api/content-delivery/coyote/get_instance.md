# get_instance

## Description

Gets a Coyote instance.

## Synopsis

```
xln-cli api content_delivery coyote get_instance [options]
```

## Options

`--organization_id (string)`

An organization id. Required.

`--instance_id (string)`

A Coyote instance ID. Required.

`--output_file (string)`

Path to output file. Written in JSON format.
