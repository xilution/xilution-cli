# list_instances

## Description

Lists Coyote instances.

## Synopsis

```
xln-cli api content_delivery coyote list_instances [options]
```

## Options

`--page_number (number)`

The page number. Required.

`--page_size (number)`

The page size. Required.

`--output_file (string)`

Path to output file. Written in JSON format.
