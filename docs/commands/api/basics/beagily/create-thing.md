# create-thing

## Description

Creates a Beagily thing.

## Synopsis

```
xln-cli api basics beagily create-thing [options]
```

## Options

`--input-file (string)`

Path to JSON formatted input file. Required.

`--output-file (string)`

Path to output file. Written in JSON format.

`--type (string)`

A thing's type. Required.
