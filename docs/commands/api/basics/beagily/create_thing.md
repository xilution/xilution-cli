# create_thing

## Description

Creates a Beagily thing.

## Synopsis

```
xln-cli api basics beagily create_thing [options]
```

## Options

`--input_file (string)`

Path to JSON formatted input file. Required.

`--output_file (string)`

Path to output file. Written in JSON format.

`--type (string)`

A thing's type. Required.
