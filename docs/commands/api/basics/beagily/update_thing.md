# update_thing

## Description

Updates a Beagily thing.

## Synopsis

```
xln-cli api basics beagily update_thing [options]
```

## Options

`--input_file (string)`

Path to JSON formatted input file. Required.

`--output_file (string)`

Path to output file. Written in JSON format.

`--thing_id (string)`

A thing ID. Required.

`--type (string)`

A thing's type. Required.
