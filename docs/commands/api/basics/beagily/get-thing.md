# get-thing

## Description

Gets a Beagily thing.

## Synopsis

```
xln-cli api basics beagily get-thing [options]
```

## Options

`--output-file (string)`

Path to output file. Written in JSON format.

`--thing-id (string)`

A thing ID. Required.

`--type (string)`

A thing's type. Required.
