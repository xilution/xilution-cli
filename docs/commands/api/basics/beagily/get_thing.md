# get_thing

## Description

Gets a Beagily thing.

## Synopsis

```
xln-cli api basics beagily get_thing [options]
```

## Options

`--output_file (string)`

Path to output file. Written in JSON format.

`--thing_id (string)`

A thing ID. Required.

`--type (string)`

A thing's type. Required.
