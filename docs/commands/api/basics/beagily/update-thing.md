# update-thing

## Description

Updates a Beagily thing.

## Synopsis

```
xln-cli api basics beagily update-thing [options]
```

## Options

`--input-file (string)`

Path to JSON formatted input file. Required.

`--output-file (string)`

Path to output file. Written in JSON format.

`--thing-id (string)`

A thing ID. Required.

`--type (string)`

A thing's type. Required.
