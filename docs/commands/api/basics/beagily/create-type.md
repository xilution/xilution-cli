# create-type

## Description

Creates a Beagily type.

## Synopsis

```
xln-cli api basics beagily create-type [options]
```

## Options

`--input-file (string)`

Path to JSON formatted input file. Required.

`--name (string)`

The type's name. Required.

`--output-file (string)`

Path to output file. Written in JSON format.
