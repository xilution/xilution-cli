# update-type

## Description

Updates a Beagily thing.

## Synopsis

```
xln-cli api basics beagily update-type [options]
```

## Options

`--input-file (string)`

Path to JSON formatted input file. Required.

`--name (string)`

The type's name. Required.

`--output-file (string)`

Path to output file. Written in JSON format.
