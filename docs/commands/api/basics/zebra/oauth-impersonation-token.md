# oauth-impersonation-token

## Description

Generates an authentication impersonation token.

## Synopsis

```
xln-cli api basics zebra oauth-impersonation-token [options]
```

## Options

`--client-id (string)`

A Xilution client's ID. Required.

`--username (string)`

A Xilution user's username. Required.
