# delete_pipeline

## Description

Deletes a Coyote pipeline.

## Synopsis

```
xln-cli api content_delivery coyote delete_pipeline [options]
```

## Options

`--organization_id (string)`

An organization id. Required.

`--pipeline_id (string)`

A Coyote pipeline ID. Required.
