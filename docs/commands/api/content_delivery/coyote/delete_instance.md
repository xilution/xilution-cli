# delete_instance

## Description

Deletes a Coyote instance.

## Synopsis

```
xln-cli api content_delivery coyote delete_instance [options]
```

## Options

`--organization_id (string)`

An organization id. Required.

`--instance_id (string)`

A Coyote instance ID. Required.
