# xilution-cli

The Xilution command line interface.
Provides command line abstractions for Xilution APIs.

## First Things First

* Run `yarn` to load dependencies

## To Verify

* Run `yarn verify`

## To Build

* Run `yarn build`

## To See Usage

* Run `yarn link` to make command executable
* Run `xln-cli --help` to see CLI usage
